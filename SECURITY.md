# Security Policy

## Reporting a vulnerability

Please **do not** open a public GitHub Issue for security vulnerabilities.

Report them privately via [GitHub Security Advisories](https://github.com/alfred-im/alfred-im/security/advisories/new).

Include:

- A description of the issue and its impact
- Steps to reproduce, if possible
- Affected components (client, Supabase migrations, bridges, etc.)

We will acknowledge receipt and work on a fix as soon as practicable.

## Supported versions

Security fixes apply to the default branch (`main`) and the latest deployed web client when relevant.
